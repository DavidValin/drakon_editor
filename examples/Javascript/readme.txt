How to run
==========

1. Install a JavaScript shell, for example 
    SpiderMonkey
    https://developer.mozilla.org/en-US/docs/Mozilla/Projects/SpiderMonkey/Introduction_to_the_JavaScript_shell
    or WebKit
    https://trac.webkit.org/wiki/JSC

2. run in the command line:
    js Demo.js
    or
    jsc Demo.js
    
Note: the script will not work with the Windows scripting host, cscript.exe
